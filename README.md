# Design And Development of a Web-Based Human Resource Portal for Automating Core HR Functions: 

•	Developed a web-based Human Resource Management Portal using HTML, CSS, JavaScript, React, Node.js, and MongoDB to automate core HR functions such as employee data management, attendance tracking, and role-based access. 

•	The system streamlines HR operations by reducing manual effort, improving data accuracy, and providing a centralized platform for managing employee records. Designed with a user-friendly interface and a secure backend to support efficient HR workflows within an organization.
